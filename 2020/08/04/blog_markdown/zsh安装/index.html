安装zsh

  • 查看是否已经安装
  • 安装 zsh: sudo pacman -S zsh
  • 安装 oh-my-zsh

  • 设为默认

  • chsh -s /bin/zsh

  1. 安装插件

  • 代码提示插件：zsh-autosuggestions

  • 代码高亮插件：

zsh-syntax-highlighting

将插件使用 git 工具 clone 到

编辑 ~/.zshrc, 修改 zsh 主题为 agnoster

  • 还可以添加 extract,z,fzf 等插件。

还可以使用网上的一个 zsh 主题：zeta-zsh, 下载到 oh-my-zsh/custom/themes 启用即可（启用之后刷新配置）！

incr也是自动补全插件

下载自动补全插件http://mimosa-pudica.net/src/incr-0.2.zsh

放到oh-my-zsh的插件库中
.oh-my-zsh/plugins/创建incr文件夹放到该文件夹中

在.zshrc中添加

在.zshrc文件末尾添加下面这行

source ~/.oh-my-zsh/plugins/incr/incr*.zsh
